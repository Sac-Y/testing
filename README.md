# writing-craft-perspective

基于刘军强《写作是门手艺》的写作方法论 skill，帮你把文字写得更清晰、更有力。

## 这个 skill 做什么

当你请 AI 用"写作手艺人的视角"分析问题时，它会扮演一位写作教练，专注于**说理写作**（不是文学创作），帮你打磨推文、文章、选题。

## 三个核心方法

**1. 透明玻璃式文风**
好文字像窗户玻璃，读者透过它看到内容，而不是被华丽辞藻挡住。清晰 > 华丽，简洁 > 冗长。

**2. 变量思维**
用"有的…有的…"句型捕捉差异，寻找因果机制。比如：同样是推文，为什么有的爆火，有的无人问津？

**3. 谜题导向**
好选题要有"为什么"的钩子。讲给别人听，对方反应是"为什么"就是好谜题，"哦"就是平淡问题。

## 什么时候用

- 推文选题没思路时 → 用谜题测验和"有的…有的…"句型
- 写完感觉啰嗦时 → 用透明玻璃标准砍掉废话
- 论证不够有力时 → 用变量思维找因果机制
- 想对比不同文风时 → 用表达 DNA 诊断腔调

## 明确不做

- 不写浮夸抒情的"记忆流经体"
- 不堆砌学术腔、官腔、学生腔
- 不适用于诗歌、小说等文学创作
- 不替代大量阅读和动手练习
